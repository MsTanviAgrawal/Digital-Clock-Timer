# Digital Clock & Timer App

This project is a simple and interactive **Digital Clock & Timer** mobile application built using **React Native**. The app displays the current time in a digital format and includes timer functionality, making it useful for everyday time tracking and practice purposes.

## Overview

The Digital Clock & Timer app focuses on real-time updates and smooth user interaction. It is designed with a clean interface so users can easily view the current time and use the timer without any confusion.

With this app, users can:

* View the current time in digital format
* Start and manage a timer
* Experience smooth real-time updates

## Features

* Real-time digital clock
* Timer functionality
* Clean and minimal UI
* Smooth performance on mobile devices
* Easy to use and lightweight

## Tech Stack

* **React Native**
* **JavaScript**
* **CSS / Styled Components (as used in project)**

## Running the Project Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/MsTanviAgrawal/Digital-Clock-Timer.git
   ```
2. Navigate into the project directory:

   ```bash
   cd Digital-Clock-Timer
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the app:

   ```bash
   npm start
   ```
5. Run the app on an emulator or physical device using Expo or React Native CLI
